# Administração

Este documento serve para o acompanhamento das aulas, projetos, entregas e avaliações da disciplina de **Administração**.

## 📊 Critérios e Instrumentos de Avaliação

A avaliação da disciplina é composta por:
- **T1**: Trabalho Escrito (20%)
- **S1**: Seminário (30%)
- **P1**: Prova (50%)

**Cálculo da Média:**
`Média = (T1 * 0.2) + (S1 * 0.3) + (P1 * 0.5)`
